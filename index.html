Access REPL Ping Test Worm

This is a demonstration tool for connecting and controlling a Bangle.js 2 via the Web Bluetooth API.
Only Chrome version 70+ on Windows or MacOS is supported.

Created as a part of ACM Research Spring 2023 at the University of Texas at Dallas
by Rishit Viral, Samuel Preston, Ben Kye, Naja-Lee Habboush, Daniel Nguyen, and Soumika Seelam.
